# Session4 Reflection

## Summary
Today, we learned how to publish our work as Reproducible Documents using Quarto. 
We then learned NumPy Arrays; even though they are restricted to a single data type, 
they are much easier to operate for mathematical calculations. 
Finally, we learned how to design an algorithm by identifying the peaks of each pulse 
to calculate the heart rate.
## Reflection
I think today’s session was fun because we learned how to process real physiological data. 
I liked the format of the session where we could discuss our own logic first before being 
instructed on the most efficient methods. This approach strengthened my understanding and 
made it much easier to digest the reasoning behind algorithm design. I believe it would be 
very worthwhile to learn different algorithms and apply them to my future work, perhaps even 
using them as inspiration for my future research topics.
